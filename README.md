# Проект: Путешествие по России

## О проекте

Учебный проект, который представляет собой карту интересных мест России
Основной стек: HTML, CSS

### Технологии

Использовала структуру Nested БЭМ
Сделала responsive (3 брейкпоинта: 728, 1024, 1280)
Mobile-first
Figma
CSS Transition
Подключаемые шрифты
